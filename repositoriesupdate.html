How to update a Repo and install sublime-text in the Kali Linux

In order to make any changes to the apt or you need a souce list

Below the codes given for a general one . suppose the repo contains the binary file .deb and the url is

http.example.com/


so by below you can add it to a list
echo "deb http.example.com" | tee /etc/apt/sources.list.d/example.list
by the above line you can add the repo and now follow the codes below

  • apt update
  • apt-get install example

    • where example is the package name

Now an examle as sublime text editor


Folowing are the codes for installing the sublime text editor built 3210 v3.xx

Teh first code will download the key
The second code will instlal apt-transport-https
The third line woluld the repo in source.list
Fourth line is just for the update
Fifth line is for installing fromt the repo

  1. wget -qO - https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add -
  2. sudo apt-get install apt-transport-https
  3. echo "deb https://download.sublimetext.com/ apt/dev/" | sudo tee /etc/apt/sources.list.d/sublime-text.list
  4. apt-get update
  5. sudo apt-get install sublime-text
